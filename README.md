# Flask-Bokeh-Htmx

Easy way to create interactive Dashboard with Flask, Bokeh and HTMX

bases on a [panel example](https://panel.holoviz.org/gallery/simple/iris_kmeans.html).

![](screencast.gif)

```bash
python -m venv .venv
pip install --no-cache-dir -r requirements.txt
./start
```

or just try Docker
